## hbtrack

#### Installation
If you trust me:
```
curl https://raw.github.com/zachallaun/hbtrack/master/hbtrack | sudo tee /usr/local/bin/hbtrack 1>&- 2>&-
```
If you don't:
```
1. clone this repository
2. add an alias to .bashrc/.bash_profile/.zshrc
```

#### Usage
```
hbtrack repo [repo ...]
```
